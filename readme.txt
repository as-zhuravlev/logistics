Запуск постгреса:
deploy_with_service.cmd - запускает службу постгреса, требует права администратора.
deploy_without_service.cmd - запускает постгрес, консольное окошко не закрывать.

Внешние зависимости: libpq.dll (libpq.a) - драйвер Postgresql лежат в \lib.

Построение:
Есть проекты для CodeBlocks, но в них не доконца настроены пути выходных файлов.
Возможно нужно будет руками подсунуть требуемые dll-ки в выходные каталоги.

Программа должна добавить по три строчки в две таблицы базы данных (скрипт создания БД \Utils\createdb.sql).

Посмотреть это можно в \Foreign\postgresql-win32-9.3.4-4\bin\PgAdmin3.exe
