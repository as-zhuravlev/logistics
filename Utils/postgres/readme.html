Overview

The folder contains bat files along with configuration files needed to run services and applications located in the ../bin-ext folder. Some temporary and log files are also created here

Apache

The command file to run Apache web server (just close the console window to stop the server):

startup-apache.bat

Apache configuration files:

httpd.conf
mime.types

GeoServer

The command file to run GeoServer (just close the console window to stop the server):

startup-geoserver.bat

The command file sets the GeoServer data directory as 

set GEOSERVER_DATA_DIR=..\..\data\geoserver_data_dir
The above mentioned Apache httpd.conf file assumes the GeoServer listening the 8084 port.
Don't forget to setup the GeoServer port value in %GEOSERVER_HOME%\etc\jetty.xml file: 
SystemProperty name="jetty.port" default="8084"

PostgreSQL

The command file to initialize the PostgreSQL database folder structure in ../../data/pgsql_data:

init-pgsql.cmd

It generates pg_config.h containing connection string that can be used by C/C++ programs. e.g.:

static const char * vtd_db_connection_string = "dbname='vtd_spatial_db' host=localhost port=5439 user='postgres' password='postgres' ";

The PGVARS.cmd file sets the configuration variables for PostgreSQL and is called by other bat files. It defines default user, non standard 5439 port, data folders etc.

The command files to startup and shutdown PostgreSQL server:

startup-pgsql.cmd
shutdown-pgsql.cmd

The following bat files can be used to run PGAdmin and psql utilities from proper folders and with proper settings:

run-pgadmin.bat
run-psql.bat
